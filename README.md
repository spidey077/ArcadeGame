# ArcadeGame
Laser Bounce Madness is a fast-paced browser arcade game built entirely with HTML, CSS, and vanilla JavaScript. It uses the HTML5 canvas for rendering and runs directly in the browser with no external APIs, frameworks, or libraries.
The player controls a diamond-shaped character, avoids bouncing enemies, fires lasers, and collects power orbs while the game gradually becomes more challenging.

# Controls

Arrow Keys – Move
Space – Shoot lasers

# Features

Built using only HTML, CSS, and JavaScript
Canvas-based rendering
Smooth player movement and shooting
Enemies with bouncing physics
Progressive difficulty increase
Power orbs with timed disappearance
Sound effects and background music
Start menu and game-over screen
Responsive canvas resizing

# How to Play

Open the game in a modern browser
Click Start Game
Survive as long as possible and maximize your score
